Public Records Request Checklist

Include these details

  • Your name and the best phone number or email for follow-up.
  • The exact meeting, record type, permit topic, or ordinance topic if known.
  • The date or date range tied to the request.
  • Whether you need a standard copy or an accessible format.

Use the public archive first

Start with the public document hub before requesting staff follow-up. The archive is the fastest route when a guide or reference file has already been posted.

When to contact the Clerk directly

Email deb.dillon@townofwiley.gov when a file is not posted yet, when you need an accessible copy, or when the exact record requires a date-specific follow-up.
