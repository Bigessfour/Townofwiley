Welcome, {{ clerkName }}

This page is the Town of Wiley clerk setup handoff for the live CMS. Use it to open the Town's Amplify Studio workspace and begin managing homepage content.

Open Studio Home Open Data Manager

What you need first

  • An AWS console login for the Town AWS account
  • Access to the Townofwiley Amplify app in us-east-2
  • Your browser signed in to the correct AWS account before opening Studio

What you will edit

  • SiteSettings for homepage title and welcome text
  • AlertBanner for emergency banner content
  • Announcement for notices
  • Event for calendar entries
  • OfficialContact for public contact cards
  • EmailAlias for private town email forwarding rules

First-time sign-in notes

  1. Open Studio Home first.
  2. If AWS asks you to sign in, use your Town AWS console account.
  3. After Studio loads, open Data Manager to edit records.
  4. Save your changes and refresh the public homepage to confirm them.

This is a setup handoff page, not a standalone CMS login portal. If access is denied, the Town AWS account still needs to grant your AWS console permissions.
