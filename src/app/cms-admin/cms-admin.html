Publishing Flow

How to update the homepage now

  1. Open Amplify Studio from the AWS console.
  2. Update the SiteSettings, AlertBanner, Announcement, or OfficialContact record.
  3. Save the record in Studio.
  4. Refresh the public homepage and confirm the change.

Model Map

Which Studio models control which parts of the homepage

  • SiteSettings: homepage title, hero copy, welcome text
  • AlertBanner: emergency banner visibility and text
  • Announcement: notice cards on the homepage
  • OfficialContact: public contact cards
